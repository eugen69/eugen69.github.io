Since the beginning of time cats were closely related to programming skills. You can not proceed and do human stuff without writing some code about cats. This app is my insight into that relationship.

Use android 3.0 and up. When you met a cat, mark its color. Also you can write a name or description.

View list of cats found per day, month or year. Each cat have timestamp, color, and, if available, name and location. Dark theme designed for better nigth cat hunting.

Other necessary things: satellites positions, data backup, statistics, kmz file containing all coordinates, and frequency diagram.
